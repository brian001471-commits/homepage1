# 주식회사 나눔랩 (Nanum Lab) & Deploy Vibe - Core AI Design Rules

# 1. Preferred Stack & Architecture
- Frontend Framework: Next.js (App Router, Tailwind CSS, TypeScript)
- Backend Stack: FastAPI (Microservices on Kubernetes/EKS)
- Data Syncer: Supabase Realtime (for ms-level data stream synchronization)
- Styling & Animations: Tailwind CSS, Framer Motion (60fps fluid motion)

# 2. Design System & Brand Identity
- Global Background (Deep Charcoal Blue): `#0B0F19` (Apply globally to body and section wrappers)
- Primary Accent (Electric Cyan): `#00F0FF` (Use for high-importance highlights, main CTA typography, and neon line-glow effects)
- Card / Tab Background (Muted Blue): `#1E293B`
- Status Colors (Industrial Standards): Emerald (Normal), Amber (Warning), Red (Danger)
- Typography: Inter or Pretendard (Heavy font-weight contrast between titles and body text, fluid sizing)

# 3. Floor-by-Floor Architecture Conventions

## [Floor 1] Main Hero Section (Inspired by Upstage & KATOZ)
- Layout: Full screen view (`h-screen`), centered typography.
- Visual: Integration of an interactive HTML5 Canvas or Framer Motion layer rendering an ambient 3D network node / flow graph topology.
- Content:
  - Main Heading: "경험에 의존하던 코팅 품질 관리,\n이제 AI 데이터로 확실하게 예측하십시오." (Highlight key expressions using `#00F0FF`, `text-5xl md:text-8xl`, `font-extrabold`).
  - Subheading: "IoT 센서와 AI 분석을 통해 생산 전 불량률을 사전에 시뮬레이션하고 최적의 배합비를 추천하는 나눔랩의 'Smart Coat-Ops' 솔루션."
- CTA Buttons:
  - Primary Button: "무료 데모 시연 신청" (Filled with Electric Cyan `#00F0FF`, dark text `#0B0F19`, `rounded-full`, with elegant glow `hover:shadow-[0_0_25px_rgba(0,240,255,0.6)]`).
  - Secondary Button: "서비스 소개서(PDF) 다운로드" (Ghost style with subtle slate border, crisp white text).

  - Inside the AI Tab, split into a 2-column layout:
    - Left Sub-column (Technical Spec Card): Muted Blue background (`bg-[#1E293B]`, `rounded-2xl`). List components like "Supabase Realtime ms-level sensor data syncing (Temperature, Humidity, RPM)", "Scikit-learn RandomForest automated QA engines", and "XAI (Explainable AI) faulty batch detection analytics".

Include industrial Traffic Light System indicators (Emerald, Amber, Red).
